INSTRUÇÕES:

1 - O codigo foi desenvolvido em linguaguem C
2 - É necessario ter o compilador DevC++ ou Code Blocks para executar o arquivo enviado
3 - Ao acessar o compilador ir em Open File e selecionar o arquivo enviado
4 - O metodo escolhido foi MidSquare
5 - Ao compilar o codigo é gerado um menu onde o usario pode optar por informar uma semente
6 - Em seguida deve ser informado a quantidade de numeros aleatorios que deseja obter
7 - Essa condição é finalizada anteriormente apenas se for gerado o numero 00
